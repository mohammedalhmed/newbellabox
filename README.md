# BellaBox Theme - ثيم بيلابوكس لمنصة سلة

ثيم احترافي وراقٍ مخصص لمتاجر التجميل والعناية بالبشرة على منصة سلة (Salla). يتميز الثيم بتصميم عصري، ألوان جذابة، وتجربة مستخدم ممتازة على جميع الأجهزة.

## المميزات الرئيسية ✨

- **تصميم احترافي وراقٍ**: واجهة مستخدم حديثة وأنيقة مناسبة لمتاجر التجميل
- **ألوان متناسقة**: نظام ألوان متدرج من الوردي والذهبي يعكس فخامة المنتجات
- **متجاوب بالكامل**: يعمل بكفاءة على جميع الأجهزة (سطح المكتب، أجهزة لوحية، هواتف ذكية)
- **سرعة عالية**: تحسين الأداء والسرعة لتجربة تصفح سلسة
- **دعم اللغات**: دعم كامل للعربية والإنجليزية
- **مكونات قابلة لإعادة الاستخدام**: تصميم نظيف وسهل الصيانة
- **تجربة تسوق محسّنة**: واجهات سهلة للبحث والتصفح والشراء

## الألوان المستخدمة 🎨

| اللون | الكود | الاستخدام |

| الوردي الفاتح | `#FCE4EC` | الخلفيات والعناصر الثانوية |

## الخطوط المستخدمة 🔤

- **الخط الأساسي**: Segoe UI, Tahoma, Geneva, Verdana (للإنجليزية)
- **الخط العربي**: Cairo, Arabic Typesetting, Simplified Arabic (للعربية)
- **أيقونات**: Font Awesome 6.4.0

## المكونات الرئيسية 🧩

### Header (الرأس)
- شعار المتجر
- قائمة التنقل الرئيسية مع قوائم منسدلة
- شريط البحث
- أيقونات الحساب والسلة
- قائمة جوال متجاوبة

### Hero Section (القسم الرئيسي)
- صورة خلفية جذابة
- عنوان ونص تحفيزي
- زر دعوة للإجراء

### Products Grid (شبكة المنتجات)
- عرض المنتجات في شبكة متجاوبة
- بطاقات منتجات مع صور وأسعار
- أيقونة المفضلة
- زر إضافة للسلة

### Features Section (قسم المميزات)
- أيقونات توضيحية
- وصف المميزات (دعم 24/7، شحن سريع، دفع آمن، منتجات أصلية)

### FAQ Section (الأسئلة الشائعة)
- تصميم أكورديون تفاعلي
- أسئلة وإجابات شائعة

### Footer (التذييل)
- معلومات المتجر والوصف
- روابط سريعة
- معلومات التواصل
- وسائل التواصل الاجتماعي
- طرق الدفع

## التثبيت والتطوير 🚀

### المتطلبات
- Node.js 14+
- npm أو yarn
- Salla CLI

### خطوات التثبيت

1. **استنساخ المستودع**
```bash
git clone https://github.com/mohammedalhmed/newbellabox.git
cd newbellabox
```

2. **تثبيت المتطلبات**
```bash
npm install
```

3. **تطوير الثيم محليًا**
```bash
npm run dev
```

4. **بناء الثيم للإنتاج**
```bash
npm run build
```

5. **نشر الثيم**
```bash
npm run publish
```

## الاستخدام 📖

### إضافة الثيم إلى متجرك على سلة

1. انتقل إلى لوحة تحكم متجرك على منصة سلة
2. اذهب إلى **تصميم المتجر** > **الثيمات**
3. ابحث عن ثيم "BellaBox"
4. انقر على **تثبيت** أو **تفعيل**
5. قم بتخصيص الثيم حسب احتياجاتك
